## 5.1.3 测试工具

### 功能测试工具

**Pytest**：
Python单元测试框架，用于后端API测试。

```python
# 示例测试用例
@pytest.mark.asyncio
async def test_create_paper(client):
    response = await client.post(
        "/api/v1/papers",
        json={"title": "测试论文", "abstract": "摘要内容"}
    )
    assert response.status_code == 201
    assert response.json()["data"]["title"] == "测试论文"
```

**Jest + React Testing Library**：
前端组件测试框架。

```typescript
// 示例组件测试
test('renders writing assistant', () => {
  render(<AIWritingPanel />);
  expect(screen.getByText('AI写作助手')).toBeInTheDocument();
});
```

**Playwright**：
端到端测试工具，模拟用户操作。

```typescript
// E2E测试示例
test('complete writing workflow', async ({ page }) => {
  await page.goto('/paper/new');
  await page.fill('[name="title"]', '测试论文');
  await page.click('button:has-text("AI生成大纲")');
  await expect(page.locator('.outline')).toBeVisible();
});
```

### 性能测试工具

**Locust**：
Python负载测试工具，模拟并发用户。

```python
# 负载测试脚本
class WebsiteUser(HttpUser):
    wait_time = between(1, 5)

    @task(3)
    def view_paper(self):
        self.client.get("/api/v1/papers")

    @task(1)
    def ai_write(self):
        self.client.post("/api/v1/ai/writing", json={
            "text": "研究背景：",
            "task_type": "continue"
        })
```

**k6**：
现代化负载测试工具，支持JavaScript脚本。

```javascript
// k6性能测试
import http from 'k6/http';
import { check } from 'k6';

export const options = {
  stages: [
    { duration: '2m', target: 100 },
    { duration: '5m', target: 100 },
    { duration: '2m', target: 200 },
    { duration: '5m', target: 200 },
    { duration: '2m', target: 0 },
  ],
};

export default function () {
  const res = http.get('http://api.scholarforge.io/health');
  check(res, {
    'status is 200': (r) => r.status === 200,
    'response time < 500ms': (r) => r.timings.duration < 500,
  });
}
```
