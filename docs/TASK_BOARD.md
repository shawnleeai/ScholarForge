## 📋 任务详情

### 1. RBAC 权限系统

**验收标准:**
- [ ] 支持 5 种预定义角色 (super_admin, institution_admin, advisor, student, reviewer)
- [ ] 支持资源级权限控制 (论文所有者、协作者、导师)
- [ ] API 接口受权限保护
- [ ] 前端界面根据权限动态显示/隐藏
- [ ] 权限变更实时生效

**技术要点:**
```python
# 权限检查示例
@require_permission("paper", "update")
async def update_paper(paper_id: UUID, data: PaperUpdate):
    ...
```

### 2. 消息通知系统

**验收标准:**
- [ ] WebSocket 实时推送
- [ ] 邮件通知模板
- [ ] 通知偏好设置
- [ ] 未读消息计数
- [ ] 支持 8 种通知类型

**技术要点:**
```typescript
// 前端使用
const { unreadCount, notifications } = useNotification();
```

### 3. 数据安全

**验收标准:**
- [ ] 论文内容 AES-256 加密
- [ ] 操作审计日志
- [ ] 自动备份策略
- [ ] 敏感操作二次确认

### 4. 性能优化

**验收标准:**
- [ ] Redis 缓存集成
- [ ] API 响应 P95 < 500ms
- [ ] 数据库查询优化
- [ ] 慢查询监控
