## 二、项目架构现状

### 2.1 技术栈

**后端**:
- Python 3.11
- FastAPI (Web框架)
- SQLAlchemy 2.0 (ORM)
- PostgreSQL/SQLite (数据库)
- Redis (缓存)
- Elasticsearch (搜索)
- Neo4j (知识图谱)

**前端**:
- React 18
- TypeScript
- Vite (构建工具)
- Ant Design (UI组件)
- Zustand (状态管理)

**基础设施**:
- Docker + Docker Compose
- Kong (API网关)
- GitHub Actions (CI/CD)

### 2.2 服务架构

```
ScholarForge
├── 前端 (端口 5173)
├── API网关 (端口 8000)
├── 微服务 (35个)
│   ├── AI服务 (8004)
│   ├── 用户服务 (8001)
│   ├── 论文服务 (8002)
│   ├── 文献服务 (8003)
│   └── ...
└── 数据库
    ├── PostgreSQL
    ├── Redis
    ├── Elasticsearch
    └── Neo4j
```
