## 数据加载方法

### 方法一：使用脚本（推荐）

**Windows PowerShell**:
```powershell
.\demo\scripts\load_demo_data.ps1
```

**Linux/Mac**:
```bash
./demo/scripts/load_demo_data.sh
```

### 方法二：直接执行SQL

```bash
psql postgresql://user:pass@localhost:5432/scholarforge -f demo/scripts/demo_scenario.sql
```
